# Thinkwell Enterprises - Offline Invoice Generator

A fast, beautiful, and **100% offline** web application to generate professional invoices directly in your browser. No backend server, no monthly fees, and no data leaving your device.

## ✨ Key Features

- **Privacy First (100% Offline):** All invoice data, client informations, and business details are securely stored within your own browser's `localStorage`. No data is ever uploaded or sent to a third-party server.
- **CSV Data Exports:** Need an external copy of your local data? You can instantly export all your saved invoices to a `.csv` file with a single click.
- **Instant PDF Downloads:** Generates beautiful, print-ready PDF invoices using your browser's native print engine. No external libraries needed!
- **Tax & Discount Support:** Fully dynamic calculation of Subtotals, Taxes/GST, Discounts, and Shipping costs out of the box.
- **Customization Options:** Easily upload your own company logo and change the visual theme of the invoice directly from the interface.

## 🚀 How to Run

Because this is a **Static Application** consisting entirely of standard Front-End web technologies (HTML, CSS, and plain JavaScript), it is incredibly easy to run or host:

### Running Locally
1. Download or clone this repository to your computer.
2. Double-click the `index.html` file to automatically open it in your default web browser.
3. Start creating invoices immediately! 

### Free Cloud Hosting
If you want to access the generator from anywhere, you can host the file easily via static hosts:
- **Netlify / Vercel:** Create a free account and drag-and-drop the main folder directly into their deployment window.
- **GitHub Pages:** Upload this project to a GitHub repository and enable "GitHub Pages" in the settings.

## 💾 Data Management & Security

* **Persistence:** If you refresh the page or close your browser, your customized settings, company info, and your invoice history will be automatically restored next time.
* **Warning:** If you clear your browser's cache/site data or use a Private/Incognito window, your saved data **will be permanently lost**. Please use the **Export to CSV** feature found in the 'Settings' tab regularly to back up your data!

- **CSS3:** Custom styles leveraging CSS variables and CSS Grid.
- **Vanilla JavaScript:** Zero dependencies, high-performance logic.
